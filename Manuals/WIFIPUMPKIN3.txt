
wifipumpkin3 use the port 53 for mount python dns server, when i try to install on Ubuntu 18.04,
i got somes error because this “port 53 is used by another process”. This problem is caused by 
systemd-resolved to solve only follow the step bellow.

sudo systemctl stop systemd-resolved

sudo nano /etc/systemd/resolved.conf
# This file is part of systemd.
#

# the Free Software Foundation; either version 2.1 of the License, or
# (at your option) any later version.
#
# Entries in this file show the compile time defaults.
# You can change settings by editing this file.
# Defaults can be restored by simply deleting this file.
#
# See resolved.conf(5) for details

[Resolve]
DNS=8.8.8.8
#FallbackDNS=
#Domains=
LLMNR=no
MulticastDNS=no
DNSSEC=no
#DNSOverTLS=no
Cache=yes
DNSStubListener=no


sudo ln -sf /run/systemd/resolve/resolv.conf /etc/resolv.conf

sudo make install

sudo python3 -m pip install .

sudo wifipumpkin3
